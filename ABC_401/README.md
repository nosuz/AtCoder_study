## A - Status Code

Python は、大小の判定で単純な大小の比較ではなく、2 つの比較演算子で挟むことで範囲にあるかを判定可能。

## C - K-bonacci

指定範囲の累計を次の項の値とし、指定された項の値を求める問題。フィナボッチ数列に似ているが、直前 2 項ではなく K で範囲が可変。

直接指定項の値を求められないので、順番に計算していく必要がある。この時、前項の値を覚えておく方法を工夫しないと計算時間制限を超えてしまう。List 型の操作は時間がかかるので、全てのデータが入る領域を確保してしまう。

また、10^9 で割ったあまりを回答という意味に注意する。
